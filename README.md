# Algoritmo através de JavaScript

Curso desenvolvido na plataforma Alura de Lógica de programação. Desenvolvendo primeiros programas com Javascript e HTML.

## ❔ adivinha_mais

Neste programa o usuário insere um número, e ao clicar no botão "compare com o número pensado" é informado se acertou ou errou o número gerado aleatoriamente. 
Para isso foi utilizado criação de botão, alertas, e as funções Math.round e Math.random.

## 🎯 alvo

Neste programa o usuário ao clicar no centro do alvo, recebe a mensagem dizendo que o acertou. 
Para isso foi utilizado as funções fillStyle, beginPath, fill e arc para o desenho do alvo, a função onclick e alerta.

## 🎏 bandeira-brasil

Neste programa é criado a bandeira do Brasil. 
Para isso foi utilizado através do canvas as funções fillStyle, fiilRect, beginPath, fill, moveTo, lineTo e arc, fazendo assim um quadrado, dois triângulos e um círculo para a criação da bandeira.

## 🎏 bandeiras

Neste programa intercalo a bandeira do Brasil e a bandeira da Alemanha. 
Para isso foi utilizado através do canvas as funções do programa bandeira-brasil, para a criação das bandeiras e a função setIntervel para intercalar suas aparições.

## ⚫ circulo_continuo

Neste programa o usuário seleciona uma cor de sua preferência e pode realizar um desenho na tela. 
Para isso foi utilizado através do canvas as funções onmousemove, onmousedown e onmouseup.

## ⚪ circulo_shift

Neste programa o usuário possui uma tela para desenho, onde através da tecla Shift pode aumentar seu pincel, e com a tecla Alt pode diminuir seu pincel. 
Para isso foi criado um evento para reconhcer sempre que a tecla Shift e Alt são pressionadas.

## 👻 creeper

Neste programa é o Creeper (personagem do Minecraft). 
Para isso foi utilizado através do canvas as funções fillStyle, fiilRect.

## 🎨 desenha

Neste programa o usuário pode selecionar uma das três cores dos quadrados apresentados e desenhar no espaço delimitado. 
Para isso foi utilizado através do canvas funções para reconhecer o movimento do mouse e seleção de cores.

## 📐 esquadro

Neste programa é criado um esquadro. 
Para isso foi utilizado através do canvas as funções fillStyle, beginPath, fill, moveTo, lineTo criando assim dois triângulos.

## 🌺 flor

Neste programa é criado três flores. 
Para isso foi utilizado através do canvas as funções fillStyle, beginPath, fill, e arc, fazendo assim círculo para formar as flores.

## ➗ fracao

Neste programa é criado uma fração. 
Para isso foi utilizado através do canvas as funções fillStyle e fillText.

## 📊 grafico-barras

Neste programa é dois gráficos em barras comparativos de nos diefrentes. 
Para isso foi utilizado através do canvas as funções fillStyle e fillText, e funções de somas para se obter seu tamanhos.

## 🍴 ingredientes

Neste programa é apresentado um alerta pedindo para informar o números de ingredientes a ser utilizado, e depois perguntado quais são os ingredientes, não aceitando repetidos. 
Para isso foi implementada a lógica utilizando alertas.

## 💟 pulsar

Neste programa é apresentado um círculo pulsando. 
Para isso foi utilizado através do canvas funções como dos programas anteriores para desenhar, círculo e a função clearRect.

## ➡️ seta-teclado

Neste programa é possível mover a bolinha da tela através das setas do teclado. 
Para isso foi utilizado através do canvas funções como dos programas anteriores para desenhar, círculo, a função clearRect e criado um evento para reconhecer as setas do teclado quando pressionadas.
